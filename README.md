## What is?

* 멀티 모듈 기반의 모놀리식 아키텍쳐로 *EDA(Event Driven Architecture)* 를 구현한 SpringBoot 애플리케이션 입니다.
  * 간단한 saga pattern 구현

* 구조화 된 모놀리식 아키텍쳐에서 간단한 *CQRS* 패턴을 구현합니다.

* *SRP* 와 *OCP* 를 중점으로 설계되었습니다.
